Microsoft Solitaire

A Kezdetek

A Microsoft a játékot a Windows 3.0 óta, 1990-től a Windows termékcsalád részeként adta ki. A játékot 1988-ban a Wes Cherry fejleszte ki. Magát a kártyapaklit a Macintosh úttörője, Susan Kare tervezte. Cherry verziója egy "boss-kulcsot" tartalmazott, amely hamis Microsoft Excel-táblázatra váltotta volna a játékot, de megkérték, hogy távolítsa el ezt a végleges kiadásból. A Microsoft a Solitaire-t az operációs rendszertől megfélemlített emberek megnyugtatására szánta, és abban az időben, amikor sok felhasználó még nem ismerte a grafikus felhasználói felületeket, hasznosnak bizonyult az egér használatának megismertetésében, például a fogd és vidd módszerrel, a kártyák mozgatásához szükséges technika. Egy Microsoft kutatás szerint a Solitaire a három leggyakrabban használt Windows-program között volt, megelőzve a Word-t és a Microsoft Excelt.

Újkor a Solitaire történetében

2012 októberében, a Windows 8 operációs rendszer megjelenésével együtt, a Microsoft kiadta a Solitaire új verzióját Microsoft Solitaire Collection néven. Ez a Microsoft Studios által tervezett játék, William Bredbeck által tervezett kinézettel fejlesztett verzió számos új funkciót vezetett be a játékba. A játék eredeti kiadásához hasonlóan William Bredbeck is azt mondta, hogy "Az újratervezés egyik szándéka az volt, hogy megismertesse a felhasználókat az új Windows 8 operációs rendszerben beépített újszerű változtatásokkal". Ez a kialakítás továbbra is használatban van a Windows 11 rendszeren belül.

Születésnapok és az új statisztikák

A Microsoft Solitaire 2015. május 18-án ünnepelte fennállásának 25. évfordulóját. Ennek az eseménynek a megünneplésére a Microsoft pasziánszversenyt szervezett a Microsoft campusán, a fő eseményt pedig a Twitchen közvetítette. A 2020-as 30. évfordulójára a becslések szerint a játéknak még mindig 35 millió aktív havi játékosa volt, és naponta több mint 100 millió játékot játszottak le a Microsoft szerint.
